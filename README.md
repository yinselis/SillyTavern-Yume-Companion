# 🌸 梦向陪伴 (Yume Companion) - SillyTavern 插件

专为 SillyTavern（酒馆）打造的梦女/梦男专属高阶陪伴插件。为你提供沉浸式的伴侣体验、无感记忆植入以及独立的情感互动空间！

## ✨ 核心功能

1. **📱 独立悬浮手账本**
   - 屏幕上新增一个可自由拖动（长按拖拽）的悬浮球！
   - 点击即可展开独立面板，不占用聊天框空间。
   - **支持自定义头像**：在扩展设置里填入网图 URL，即可把悬浮球换成老公/老婆的头像！
   
2. **🩸 高阶生理期与档案追踪**
   - 绝密档案：填入你的身高、体重、MBTI与今日心情。
   - **生理期算法**：只需填入上次日期和周期，插件会自动计算你正处于经期第几天，或者距离下次还有几天。
   - **无感记忆同步**：上述所有身体状态都会在后台“静默”同步给 AI。当你在生理期或心情差时，AI 会主动在对话里心疼你、给你提供情绪价值！

3. **💕 纪念日系统**
   - 填入你们在一起的日期，面板顶部会自动展示你们相伴的天数。

4. **🫂 专属陪伴互动**
   - 突然想TA了？在“互动”面板点击 `戳戳TA`、`求抱抱`、`哄睡觉`、`倾诉`。
   - 插件会在后台向 AI 发送指令，随后直接弹出一个精美的提示框，展示 TA 对你的专属回应，**完全不会弄乱当前的聊天记录**！

5. **💌 后台信笺系统**
   - 有些话不好意思在聊天里说？在“信笺”面板写下留言并悄悄寄出。
   - 几秒钟后，你的伴侣会写一封长长的回信送达你的手账本中。所有的信件都会以手账的形式永久保存。

## 📥 安装与使用指南

1. **安装插件**：
   - 打开酒馆界面。
   - 点击顶部导航栏的 **积木图标 (Extensions)**。
   - 找到最上方的 **安装扩展 (Install Extension)** 按钮。
   - 复制本 GitHub 仓库的链接并粘贴：`https://github.com/你的名字/你的仓库名`。
   - 点击安装，完成后**按 F5 刷新页面**。

2. **如何打开设置？**
   - 刷新后，右下角会出现悬浮球。
   - 在酒馆左侧的“积木”扩展列表中，找到 **“🌸 梦向陪伴 基础设置”**，可以在这里更换悬浮球头像和开关悬浮球。

3. **常见问题：我刷新了怎么没看到悬浮球？**
   - 如果遇到浏览器缓存问题，请在酒馆页面按 `Ctrl + F5` 强制刷新缓存。手机端请杀掉后台重新打开浏览器。
