# 🌸 梦向陪伴 (Yume Companion) - SillyTavern 插件

专为 SillyTavern（酒馆）打造的梦女专属高阶陪伴插件。为你提供沉浸式的伴侣体验、无感记忆植入、丰富的情感互动空间，以及完全独立于主聊天的手账系统！

## ✨ 核心亮点与功能

### 1. 🎨 沉浸式专属手账 UI
- **自由悬浮球**：屏幕上新增一个可长按拖拽的悬浮球，支持自定义大小和替换为TA的头像。
- **8 大内置主题**：暗夜樱花、晨曦白纸、抹茶信笺、落樱缤纷、哥特暗夜、深海之息、霓虹赛博、橘色落日，随心切换。
- **透明度与自定义 CSS**：支持调节面板透明度，甚至开放了完整的 CSS 接口，供你打造独一无二的界面。
- **千人千面**：所有手账数据（信件、日记、档案等）**与当前聊天的角色深度绑定**，切换角色自动切换对应的手账本！

### 2. 🧠 无痕记忆注入 & 悄悄话机制
- **绝密档案与生理期追踪**：填入你的生日、MBTI、今日心情与生理期数据。插件会自动计算纪念日天数和经期状态，并**静默注入**到酒馆主聊天的最高优先级提示词中，让 TA 在日常聊天中自然地关心你。
- **📌 记事本与 25% 悄悄话机制**：在记事本写下你想让 TA 记住的备忘录。当你在主聊天框发送消息时，系统有 **25% 的概率**向 TA 发送一条隐秘指令，让 TA 像突然想起来一样，主动在回复中提及你记事本里的内容！

### 3. ✨ 进阶：TA 的主动惊喜 (省钱可控)
- 扩展设置中提供**“允许TA主动制造惊喜”**的独立开关。
- **开启后**：TA 会在聊天时，概率触发【突发长信】（悄悄塞一封信到你的手账里）和【自动观察记事】（根据聊天记录写下 TA 此刻的内心独白）。
- **关闭后**：插件绝对不会在后台主动发起任何请求，完全不浪费你的 API 额度，仅保留基础的档案读取功能。

### 4. 💌 丰富的情感互动模块
- **💬 即时互动**：摸摸头、求抱抱、牵手手、缠绵... 点击按钮，TA 会立刻给出专属的动作与情话反应，**完全不污染主聊天记录**。
- **💌 延时信笺**：写下留言并封装寄出。TA 不会立刻回复，而是在你们聊了几句之后，结合刚才的聊天内容给你写一封回信。
- **📖 交换日记**：记录你的日常，并允许 TA 偷看留言；或者点击“偷看TA的日记”，让 TA 为你写下一篇专属的私密日记。支持论坛式的盖楼评论！
- **📇 字卡系统**：像抽卡一样与 TA 互动！支持一键从过往聊天记录中**AI 提取** TA 曾说过的动人情话，存入字卡库。

### 5. 🔌 独立后台 API (强烈建议)
- 嫌后台互动消耗主 API（如 Claude）太贵？插件支持配置**独立的 API 通道**（如便宜的 GPT-4o-mini 或本地模型）。
- 配置后，信笺、日记、互动、字卡提取等所有后台操作都会走独立通道，**既能大幅省钱，又能彻底避免污染酒馆主聊天的上下文！**

## 📥 安装与使用指南

1. **安装插件**：
   - 打开酒馆界面。
   - 点击顶部导航栏的 **积木图标 (Extensions)**。
   - 找到最上方的 **安装扩展 (Install Extension)** 按钮。
   - 复制本 GitHub 仓库的链接并粘贴。
   - 点击安装，完成后**按 F5 刷新页面**。

2. **如何打开设置？**
   - 刷新后，屏幕右侧会出现悬浮球。
   - 在酒馆左侧的“积木”扩展列表中，找到 **“🌸 梦向陪伴 设置”**，可以在这里配置独立 API、修改悬浮球样式、开关主动惊喜功能，以及查看**详细的使用手册**。

3. **数据安全**：
   - 插件在“档案”页底部提供了**导入/导出数据**功能，换设备或重装酒馆前，请务必导出你的手账数据以防丢失。

## 💡 常见问题

**Q: 我刷新了怎么没看到悬浮球？**
A: 如果遇到浏览器缓存问题，请在酒馆页面按 `Ctrl + F5` 强制刷新缓存。手机端请杀掉后台重新打开浏览器。

**Q: 为什么我点互动/写信，TA 回复的是“（生成失败）”？**
A: 请检查你是否在扩展设置中填写了错误的独立 API。如果留空，插件会默认使用酒馆当前的主 API；如果填写了，请确保 URL（通常以 `/v1` 结尾）和 Key 完全正确，并点击“保存并测试”验证连通性。

**Q: 点击手册或下拉菜单时，酒馆的侧边栏自动关了怎么办？**
A: 插件已内置防穿透机制，正常点击弹窗内部不会关闭侧边栏。如果依然误触，请尽量在关闭酒馆侧边栏的情况下，通过点击悬浮球来沉浸式使用手账。
